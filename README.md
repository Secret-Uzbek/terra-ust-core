> Specification core for the Universal Semantic Translator: the integration
> matrix, conceptual interfaces, and the core agreement layer between theory,
> implementation, and deployment.

## Layer role

This repository is a specification-core layer.

It should hold:

- core UST specifications;
- integration matrices and conceptual interfaces;
- readable documentation linking theory and implementation;
- release and citation surfaces for the specification layer.

It should not become:

- a helper-generated shell;
- a fake DOI placeholder surface;
- a donor governance repository;
- a random archive of unrelated practical artifacts.
